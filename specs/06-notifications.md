# 🔔 الإشعارات (Notifications)

## للعميل عبر واتساب

```javascript
// تأكيد الطلب
const confirmMsg = `
مرحباً ${name} 👋
تم استلام طلبك رقم #${orderNumber} ✅
الإجمالي: ${total} جنيه
سيتم التواصل معك قريباً 🚗
`

// الشحن
const shippingMsg = `
طلبك رقم #${orderNumber} في الطريق إليك 🚗
`

// ⭐ تذكير السلة المتروكة (✅ بعد ساعة — محسومة في كل مكان + كود خصم 10%)
const abandonedCartMsg = `
مرحباً! 👋
لسه عندك منتجات في سلتك 🛒
إتمام طلبك: ${cartUrl}
`

// ⭐ طلب تقييم بعد التسليم (بعد يومين)
const reviewMsg = `
وصلك طلبك كويس؟ 😊
يسعدنا رأيك في المنتجات: ${reviewUrl}
`

// ⭐ إشعار نقاط الولاء
const loyaltyMsg = `
عندك ${points} نقطة، استخدمها في طلبك الجديد 🎁
`
```

---

## لصاحب المتجر

```
✓ Browser Notification + صوت تنبيه عند كل طلب جديد
```

## جدول زمني للإشعارات

| الحدث | التوقيت | القناة |
|---|---|---|
| طلب جديد — للعميل | فوري | واتساب |
| طلب جديد — للمتجر | فوري | Browser + صوت |
| تغيير حالة الطلب | فوري | واتساب |
| تأكيد التسليم (صورة) | عند التسليم | واتساب |
| طلب تقييم | بعد يومين | واتساب |
| تذكير السلة المتروكة | ✅ بعد ساعة | واتساب |
| إشعار نقاط الولاء | بعد كل طلب | واتساب |
