# 🎨 Design System

```css
:root {
  --bg:          #FAFAF8;
  --bg-2:        #F5F0EB;
  --text:        #1C1C1C;
  --text-muted:  #888888;
  --border:      #E5E0DA;

  /* Accent — بيتغير لكل عميل */
  --accent:      #C9956A;   /* ذهبي دافئ — مناسب للهدايا والإكسسوارات */
  --accent-light:#FDF3EB;

  /* ⭐ ألوان Urgency */
  --danger:      #E53E3E;   /* "باقي 3 قطع بس" */
  --success:     #38A169;   /* "متاح" */
}

/* خط عربي احترافي */
font-family: 'Cairo', sans-serif;

/* Border Radius */
--r-card:   12px;
--r-btn:    10px;
--r-input:  8px;
```

## ملاحظات التخصيص لكل عميل

- `--accent` هو المتغير الوحيد اللي بيتغير لكل عميل
- اللوجو + اسم المتجر + الصور بيتغيروا مع كل عميل
